# scRNAseq-QC-Pipeline

A reproducible quality control pipeline for single-cell RNA sequencing (scRNA-seq) data analysis using R and Seurat.

## Overview

Quality control (QC) is one of the most important steps in single-cell RNA sequencing analysis. Poor-quality cells, empty droplets, and technical artifacts can affect downstream biological interpretation.

This project implements a standard QC workflow for 10X Genomics scRNA-seq datasets.

## Workflow

The pipeline includes:

- Loading 10X Genomics data
- Quality metrics calculation
- Visualization of QC features
- Filtering low-quality cells
- Data normalization
- Generation of QC reports

## Tools

- R
- Seurat
- ggplot2
- dplyr

## Input

The pipeline accepts raw single-cell RNA sequencing data generated by 10X Genomics.
