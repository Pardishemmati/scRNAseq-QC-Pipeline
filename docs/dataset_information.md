# Dataset Information

## Dataset

10X Genomics Peripheral Blood Mononuclear Cells (PBMC) 3k dataset.


## Description

This dataset contains single-cell RNA sequencing data from approximately 3,000 peripheral blood mononuclear cells.


## Source

10X Genomics public datasets.


## Data Type

- Single-cell RNA sequencing
- 10X Genomics platform


## Input Format

The pipeline expects raw count matrices:

- matrix.mtx
- features.tsv
- barcodes.tsv


## Usage

The raw data should be placed in:

data/raw/
